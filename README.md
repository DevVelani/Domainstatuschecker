# Domain Status Checker
Checks the status codes of websites (domain) and their subdomains, to see if they are Online or Offline on the Internet.

## Requirements 

```bash
┌─[root@kali]─[/]
└──╼ apt install jq
```

## Usage 

```bash 
sudo chmod 755 ./sitestatus.sh
```
```bash
┌─[root@kali]─[/]
└──╼ ./domainstatuschecker.sh <domain>
```
